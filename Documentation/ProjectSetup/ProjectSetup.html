What is Slopathon?

Slopathon is a chaotic 2D browser game. The player controls the Red Bull astronaut — a winged spaceman — navigating a hostile environment with deliberately unstable physics and maximum visual noise.

How do I run it?

No build step required. Open index.html directly in any modern browser.
